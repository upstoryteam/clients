# Growth brief — quality review

Run this checklist **after** the page matches `FORMAT.md`. Use it before publishing or sending the link. Flag failures by section; fix copy or artifacts, not the arc.

## Outcome headline (quantified)

- [ ] Headline states a **measurable outcome** (conversion, completion, activation, payment rate, etc.), not a vibe or strategy summary.
- [ ] Includes a **number or defensible range** tied to that outcome (`by about 10%`, `+5–10 pp`, `within 30 days`).
- [ ] Framed as **Upstory can help achieve it** (“We believe we can help…”), not “here is what you must do.”
- [ ] You could name the metric in a dashboard without rewriting the sentence.

**Fail examples:** “Stake is at an inflection point.” / “Improve the onboarding experience.” / “Unlock growth on UMoveFree.”

## Insight (`Our read`)

- [ ] Supports the headline outcome; does **not** repeat the same number unless necessary.
- [ ] **Unique to this client** (swap company name and it should read false).
- [ ] Opportunity framing (“opportunity to create value”), not empty hype (“exciting time,” “real opening”).
- [ ] Max 3 sentences; bridges to “The opportunities below…”

## Voice and tone

- [ ] **Not patronizing** (no “you may not realize,” no explaining their business back to them).
- [ ] **Not condensing** (no slide-deck fragments cramming three ideas into one line).
- [ ] **Peer, not auditor** (senior designer who looked at the product, not a rubric).

## AI phrasing (hard fail)

Scan the full page for these patterns. Rewrite any hit.

### Punctuation and structure

- [ ] **No em dashes** in page copy. Use periods, commas, or a second sentence.
- [ ] **No contrast pivots:** “it’s not X, it’s Y” / “not X but Y” / “X, not Y” at the end of a clause.  
  **Fail:** `Cash Back activation within a week of first payment, not a month out.`  
  **Fix:** `Cash Back activation within seven days of first payment.`
- [ ] **No trailing “not …”** corrections (`not generic`, `not a month out`, `not buried in settings`). State what you want positively.

### Banned words (page copy)

- [ ] No **actually**, **real** (except the hyphenated category **real-world** in internal docs only; avoid on the page), **something real**, **a real [noun]**, **genuinely**, **truly** as emphasis.
- [ ] No banned buzzwords from `SKILL.md` (leverage as noun, synergy, unlock, ecosystem, best-in-class, deep dive, move the needle, etc.).
- [ ] No generic AI transitions (“In today’s landscape,” “It’s worth noting,” “This is where X comes in”).
- [ ] No closer fluff (“happy to chat,” “if helpful,” “would love to”).
- [ ] The word **trust** does not appear in external-facing copy.

## Specificity

- [ ] Names a **product surface, channel, cohort, or moment** (not “the user journey” alone).
- [ ] Headline number ties to something defensible from the CSV row (or labeled conservative in measure bullet 3).
- [ ] Opportunities map to **Outreach Insight** and **Hiring Focus**, not generic trust-type boilerplate pasted verbatim.

## Artifacts

- [ ] Each artifact **adds information** the prose does not already say (if you delete the visual, you lose something).
- [ ] No filler visuals “because block 2 should have a funnel.”
- [ ] Journey steps are **specific labels** for this company’s flow, not placeholders (`Step 1`, `Onboarding`).
- [ ] Funnel percentages are **illustrative** when not from data, and labeled as such.
- [ ] Artifacts use **full column width** — nothing looks artificially narrow or wrapped early.

## Structure and layout

- [ ] Exactly six blocks in order: header → headline → insight → opportunities → ways to measure → pitch.
- [ ] No extra sections (fit essay, second hero, solution essays).
- [ ] `Our read` layout: lightbulb + eyebrow on one row; body left-aligned below.
- [ ] Opportunity spacing reads comfortable (not cramped between `01` / `02` / `03`).

## Measurement block

- [ ] Three bullets only; same **primary metric** as the headline.
- [ ] Bullet 1 = north-star + window; bullet 2 = cohort/isolation; bullet 3 = readout/threshold.
- [ ] Lead line matches default unless there is a deliberate, reviewed reason to change it.

## Pitch

- [ ] Upstory closer copy is **verbatim** locked text.
- [ ] Client value ends above the closer; pitch does not repeat opportunity content.

## Measurement bullets

- [ ] Each bullet is **positive** (what you would track), not a contrast pair (`X, not Y`).
- [ ] Same primary metric family as the headline.

## Quick read-aloud test

Read the page aloud in under 90 seconds. If anything sounds like ChatGPT (contrast tails, “actually,” “real,” em dashes), fix it before ship.
